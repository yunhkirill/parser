# Парсер

## Пример использования:
```python
from parser import Parser

parser = Parser()
netlist, reports = parser.load_netlist_from_file("test.netlist")
parser.save_netlist_to_file("new.netlist", netlist)
```

## Верхнеуровневое описание парсера:
Парсер - это программа, которая анализирует исходный текст формата нетлиста, проверяет его
соответствие грамматическим правилам и преобразует последовательно в Parse Tree, AST и
объектную модель схемы (блоки, пины, инстансы, сети). Кроме того, парсер обеспечивает
обратное преобразование объектной модели в текстовый формат нетлиста для сохранения
изменений.

## Алгоритм работы Parser
При запросе от редактора или чекера на чтение файла:
1. Чтение файла
2. Разбиение текста на токены с помощью ANTLR4 (проверка синтаксиса) и построение Parse
Tree
3. Последовательный обход всех блоков в нетлисте (Parse Tree) и построение AST
4. Преобразование абстрактного синтаксического дерева в объектную модель
5. Передача объектной модели
При запросе от редактора на сохранение схемы:
1. Преобразование объектной модели в текстовый формат
2. Сохранение файла
